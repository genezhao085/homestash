# HomeStash 家庭储物管家 - 产品需求文档

## 产品概述
HomeStash 是一个跨平台移动应用，用于管理家庭储物。通过拍照记录物品信息，配置收纳空间层级，实现物品的快速查找和分类管理。

## 目标用户
- 家庭用户：管理日常生活物品
- 收纳爱好者：优化储物空间利用
- 搬家/装修用户：记录物品位置

## 验收标准
- [ ] 可以拍照或从相册选择照片记录物品
- [ ] 支持创建多级收纳空间（房间→柜子→抽屉→箱子）
- [ ] 物品可以关联到具体空间
- [ ] 支持按名称、分类、位置搜索
- [ ] 所有数据离线可用
